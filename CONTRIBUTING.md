# Contributing to hello-world

Thanks for your interest in contributing! This repository is a small practice project. Here's how to get involved:

1. Check GitHub Discussions for existing questions or proposals: https://github.com/HobbyAK47/hello-world/discussions
2. If you found a bug or want a new feature, open an Issue using the templates in .github/ISSUE_TEMPLATE/.
3. Pick a "good first issue" or ask in Discussions for guidance.
4. Fork the repo and create a branch for your change:
   - git clone https://github.com/HobbyAK47/hello-world.git
   - git checkout -b my-feature
5. Make small, focused commits with clear messages, push, and open a Pull Request. Link the PR to any related issue.
6. A maintainer will review your PR. Aim to include tests or a short description of how to run your change if applicable.

Code style and tests
- Keep changes small and focused. Add a short note in the PR about how you tested your change.

Maintainer response time
- I try to reply within 48 hours. If you don't hear back, please bump the Discussion or issue politely.

Thank you for helping improve this project!
