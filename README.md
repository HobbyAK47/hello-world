# hello-world

This repository is for practicing the GitHub flow.

## Communication

- Discussions: Use GitHub Discussions for general questions and community conversation: https://github.com/HobbyAK47/hello-world/discussions
- Issues: Use Issues for bug reports and feature requests. Please use the templates in .github/ISSUE_TEMPLATE when opening an issue.
- Pull Requests: Open a Pull Request to propose changes. Link PRs to issues when applicable.

Response expectations: I try to reply within 48 hours.

See CONTRIBUTING.md for how to contribute and CODE_OF_CONDUCT.md for expected behavior.
